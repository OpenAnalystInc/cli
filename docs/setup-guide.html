⚡ Switch Models Mid-Conversation — Session Persists

The game-changer: switch between any LLM provider mid-conversation. Your session ID, chat history, tool results, and context carry over seamlessly. No other CLI does this.

  Terminal
> /model
  Available models (fetching from providers...)

  ● OpenAnalyst
    openanalyst-beta, openanalyst-pro
  ● OpenAI
    gpt-4o, gpt-4.1, o3, o4-mini (+12 more)
  ● xAI
    grok-3, grok-3-mini
  ○ Anthropic (run: openanalyst login)

> Explain this codebase                     # openanalyst-beta answers

> /model gpt-4o
  Model updated
  Previous         openanalyst-beta
  Current          gpt-4o
  Provider         OpenAI
  Session          persisted (3 messages)    # conversation carries over

> Now fix the bug you found                 # gpt-4o sees full history

> /model grok
  Provider         xAI
  Session          persisted (5 messages)    # same session ID

> /model sonnet
  Session          persisted (7 messages)    # 4 providers, 1 conversation

02

⚙ Configure

Set your API key. The CLI auto-detects the provider from your model or key.

  ~/.zshrc
# Set one or more provider keys
export OPENANALYST_AUTH_TOKEN="your-key"      # OpenAnalyst
export ANTHROPIC_API_KEY="sk-ant-..."        # Claude
export OPENAI_API_KEY="sk-..."              # GPT / Codex
export OPENROUTER_API_KEY="sk-or-..."       # OpenRouter
export XAI_API_KEY="xai-..."               # Grok
export GEMINI_API_KEY="AIza..."             # Gemini
export BEDROCK_API_KEY="..."               # Bedrock
PowerShell
$env:OPENANALYST_AUTH_TOKEN = "your-key"      # OpenAnalyst
$env:ANTHROPIC_API_KEY = "sk-ant-..."        # Claude
$env:OPENAI_API_KEY = "sk-..."              # GPT / Codex
$env:OPENROUTER_API_KEY = "sk-or-..."       # OpenRouter
03

▶ Start Using

  Terminal
$ openanalyst
# Connected · Model: openanalyst-beta

$ openanalyst "explain this codebase"
# One-shot prompt

$ openanalyst --model gpt-4o "summarize this repo"
# Use OpenAI

$ openanalyst --model sonnet "fix the bug in main.rs"
# Use Claude

$ openanalyst init
# Create project config (OPENANALYST.md)

⌨ Command Reference

All commands work identically regardless of which LLM provider you use — OpenAnalyst, Claude, OpenAI, Grok, or OpenRouter. Same skills, same tools, same slash commands.

CLI Commands

CommandDescription
openanalystStart interactive REPL session
openanalyst "prompt"One-shot prompt, then exit
openanalyst prompt "text"Explicit prompt mode (supports --output-format json)
openanalyst --model MODELUse a specific model (auto-detects provider)
openanalyst --resume FILEResume a saved session
openanalyst initCreate OPENANALYST.md and project config
openanalyst loginInteractive provider picker — select provider, enter API key, test connection, save credentials
openanalyst logoutClear all saved provider credentials
openanalyst agentsList configured agents
openanalyst skillsList available skills
openanalyst --versionPrint version info

✓ Features

Cross-Provider /model Switch
Switch from OpenAnalyst to GPT to Grok to Claude mid-conversation. Session ID, history, and context persist across providers.
Live Model Discovery
/model fetches available models from each provider's API in real-time. No hardcoded lists.
Multi-Provider Login
Run openanalyst login multiple times to add keys for different providers. All keys loaded on startup.
OPENANALYST.md
Project-specific AI instructions. Detected automatically in directory tree.
Session Persistence
Save, resume, list, and export conversation sessions.
Streaming Output
Real-time streaming with syntax-highlighted Markdown rendering.
Permission System
read-only, workspace-write, or danger-full-access modes.
Git Integration
Commit, branch, PR, issue creation — all from slash commands.
Config Hierarchy
.openanalyst.json → .openanalyst/settings.json → settings.local.json
Plugin System
Install, enable, and manage custom tool plugins.
Cost Tracking
Per-turn and cumulative token usage and cost display.
Cross-Platform
Native binaries for macOS, Linux, and Windows.
Full Ratatui TUI
Scrollable chat, inline tool cards, status bar, vim-mode input, permission dialogs. Default mode.
Multi-Agent Orchestrator
Spawn parallel sub-agents with independent tool permissions and real-time status tracking.
Multimedia Commands
Generate images, transcribe audio, text-to-speech, image analysis, Mermaid diagrams.
7 LLM Providers
OpenAnalyst, Anthropic, OpenAI, Google Gemini, xAI, OpenRouter, Amazon Bedrock.
