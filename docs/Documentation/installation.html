Installation

Install OpenAnalyst CLI on macOS, Linux, or Windows in under a minute.

System Requirements

RequirementDetails
OSmacOS 12+, Linux (glibc 2.31+), Windows 10/11
TerminalAny terminal with UTF-8 and 256-color support
Disk~25 MB (single binary)
RuntimeNone — native binary, no Node/Python required

Install

Install via the public script:

curl -fsSL https://raw.githubusercontent.com/OpenAnalystInc/cli/main/install.sh | bash

Or download a native macOS binary directly from the public release assets:

# Apple Silicon
curl -L https://github.com/OpenAnalystInc/cli/releases/latest/download/openanalyst-aarch64-apple-darwin -o openanalyst
chmod +x openanalyst
sudo mv openanalyst /usr/local/bin/openanalyst

# Intel
curl -L https://github.com/OpenAnalystInc/cli/releases/latest/download/openanalyst-x86_64-apple-darwin -o openanalyst
chmod +x openanalyst
sudo mv openanalyst /usr/local/bin/openanalyst

The full OpenAnalyst CLI with TUI is published in the public repo releases at OpenAnalystInc/cli/releases/latest.

Install via the public script:

curl -fsSL https://raw.githubusercontent.com/OpenAnalystInc/cli/main/install.sh | bash

Or download a native Linux binary directly:

# x86_64
curl -L https://github.com/OpenAnalystInc/cli/releases/latest/download/openanalyst-x86_64-unknown-linux-gnu -o openanalyst
chmod +x openanalyst
sudo mv openanalyst /usr/local/bin/openanalyst

# ARM64
curl -L https://github.com/OpenAnalystInc/cli/releases/latest/download/openanalyst-aarch64-unknown-linux-gnu -o openanalyst
chmod +x openanalyst
sudo mv openanalyst /usr/local/bin/openanalyst

All Linux binaries are published from the public repo release page: OpenAnalystInc/cli/releases/latest.

Tip
After installation, run openanalyst --version to verify.

Open PowerShell as Administrator:

irm https://raw.githubusercontent.com/OpenAnalystInc/cli/main/install.ps1 | iex

Or download a native Windows binary directly:

# x64
New-Item -ItemType Directory -Force "$HOME\.openanalyst\bin" | Out-Null
Invoke-WebRequest -Uri "https://github.com/OpenAnalystInc/cli/releases/latest/download/openanalyst-x86_64-pc-windows-msvc.exe" -OutFile "$HOME\.openanalyst\bin\openanalyst.exe"

The Windows install script is served from the public repo, and the matching native binaries are published at OpenAnalystInc/cli/releases/latest.

Install from npm:

npm install -g @openanalystinc/openanalyst-cli

Backend-only deployment:

npm install -g @openanalystinc/openanalyst-cli
openanalyst --notui
openanalyst --serve 8080

Use this when you want the hosted session API on a server without launching the TUI.

Info
This is the recommended server deployment path when you want a hosted OpenAnalyst backend with no interactive TUI attached.
Public Binary Source

The live docs, install scripts, and downloadable binaries are all published from the public repository: OpenAnalystInc/cli. For the full OpenAnalyst CLI with TUI, use the latest release assets or the install scripts above.

Verify Installation

# Check the version
openanalyst --version
# Output: OpenAnalyst CLI v2.0.3

# Check logged-in providers
openanalyst whoami
Default Launch Modes

openanalyst starts the full TUI by default. Use openanalyst --notui for backend-only CLI usage or openanalyst --serve to expose hosted HTTP session endpoints from a server.

Updating

Re-run the same install script to update to the latest version. Your configuration and sessions are preserved.

# macOS / Linux
curl -fsSL https://raw.githubusercontent.com/OpenAnalystInc/cli/main/install.sh | bash

# Windows
irm https://raw.githubusercontent.com/OpenAnalystInc/cli/main/install.ps1 | iex

Uninstalling

Remove the binary and optionally clean up config:

# Remove binary
rm /usr/local/bin/openanalyst

# Remove config (optional)
rm -rf ~/.openanalyst
